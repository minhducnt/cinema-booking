### Installation

- Easy Booking requires [Node.js](https://nodejs.org/) to run.

Set environment variables

```sh
Create a .env file in your backend and client folder
See the Environment Variables
```

Start the server

```sh
cd backend
$ yarn
$ yarn setup
$ yarn db:init
$ npm run
```

Start the client

```sh
cd client
$ npm install
$ npm start
```
